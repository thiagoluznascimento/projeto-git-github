git init = inicia um repositório.
git status = para ver o status e versões do que foi add no git
git add "nome do arquivo" = para add no git 
git . = add todos os aquivos de uma vez só.

git commit -m "Mensagem de primeira versão" = para enviar para o github - nuvem.

git push = envia o códg para o github

git remote add origin link do github = define para onde o cód está sendo enviado.

git push --set-upstream origin master = foi usado para fazer o login no git-hub e selecionar a branch principal (master)

git reflog = para estar visualizando as versões dos arquivos.

git reset --hard e o ID que vai estar na frente da versão que vc queira voltar, isso no git localmente.
	(Para encontrar o ID, temos que rodar o comando reflog para estarmos visualizando as versões dos arquivos)
